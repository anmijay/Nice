
## 一、关于 Nice：

Nice 是轻量的CSS基础代码库，没有一味的重置，而是 `注重各浏览器基础表现` ，减小开发难度。

## 二、仓库目录结构：
    
    ---
     |---- build/         nice.css的分离及开发备注
     |---- ext/           nice.css的扩展目录，实现兼容的表单与按钮美化
     |---- test/          测试文件
     |---- nice.css       包含`重置`、`表单`、`HTML5` 的基础样式
     |---- readme.md
